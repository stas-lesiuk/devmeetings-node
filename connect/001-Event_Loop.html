Event Loop

W poprzednim sprincie decydowaliśmy czy użyć metody fs.readFile czy fs.readFileSync. Jaka dokładnie jest między nimi różnica?

Node.js jest jednowątkowy. Operacje, które mogą zająć dużo czasu są delegowane i kod jest wznawiany po ich zakończeniu.

Realizowane jest to przez EventLoop - odpowiedź na problem C10K.
